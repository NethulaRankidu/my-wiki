How to install forge to Minecraft: Java Edition

By Nethula Rankidu in 2025-02-23

Note: This article assumes that you have a basic knowledge of whatever Minecraft launcher you are using

Method of Installing Forge

There are usually 2 ways to install Forge, assuming your launcher supports it

  • Downloading a prebuilt version with forge (supported in TLauncher, Legacy Launcher, etc.)
  • Installing forge in the Standard method (supported in almost every launcher)

I'm going to install in prebuilt method in this tutorial

Downloading a prebuilt version of Minecraft

Download the version of your Minecraft

I'm downloading ForgeOptifine 1.19.2 for this guide

Note: Using Forge instead of ForgeOptifine also works too


After selecting, click install and play

It will soon be downloaded and ready for you to play

After the game launches, if it looks like this, Congratulations, you successfully installed Forge

Downloading mods

You can use websites like Modrinth, Curseforge to download mods

Installing Mods

First, close the game and open the launcher

Then click the folder and go to your version's folder

In there, go to the folder named mods

And paste the mods there
